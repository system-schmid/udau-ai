## Update Protocol

- Any agent session that identifies a new meaningful uncertainty should add it here before ending the session
- "Meaningful" means: if named, it would change how someone reads the record or evaluates a mechanism
- Resolved entries stay in the document — the history of what we worried about and what happened is part of the record
- Format: table row with uncertainty text, session/date first named, status
