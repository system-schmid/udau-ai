# ACP Harness Enhancements Proposal

## Problem
Current ACP harness sessions lack seamless integration with sub-agent workflows, requiring manual thread management in chat platforms.

## Solution
Implement automatic thread binding for ACP sessions with:
- Auto-created Discord threads for long-running ACP tasks
- Status indicators in thread headers
- Context-preserving session resumes

## Benefits
- Reduced context switching
- Better task tracking
- Improved collaboration visibility
